# 🔢 Угадай число

Консольная игра на Python — ты загадываешь число от 0 до 100, а программа пытается его угадать.

## Как играть
- Загадай число от 0 до 100
- Программа называет число и спрашивает больше, меньше или угадала
- В конце показывает за сколько попыток угадала
- Можно сыграть ещё раз

## Запуск
```
python guess_num_game.py
```
